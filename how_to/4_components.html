The previous section showed how to create a component in a very short way. This is accomplished by using meaningful default values. However, the full description of a component is quite complex. It has the following properties:

  • BufferInfo - Contains user data and a list of buffers that describe where to find the data and how long the dataset is (link).

  • Channels[] - A list of channels.

  • DisplayInfo - Specifies how to display the data in FAMOS.

  • EventReference - Specifies which events belong to this component (link).

  • Name - Returns the name of the first channel found.

  • PackInfo - Specifies the data type and how the data are packed.

  • TriggerTime - A type to define when the data were created / recorded (link).

  • Type - The component's type.

  • XAXisScaling - Scales the x-axis.

  • ZAXisScaling - Scales the z-axis and specifies if and how the axis is segmented.
